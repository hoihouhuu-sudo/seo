# seo
data-analysis-practice

## 프로젝트 목적
본 프로젝트는 ai/dx 전문가교육 실습에서 진행한
데이터 분석에서 실습한 내용을 정리하기 위해 진행하였다.


## 사용데이터
서울특별시 공공자전거 일별 대여건수
영화 장르별 평점 비율

## 분석 및 시각화 내용
-날짜별 따릉이 대여건수 변화 (선그래프)
-따릉이 일별 대여건수 분포 (막대그래프)
-따릉이 일별 대여건수 및 7일 이동평균(선그래프)
-요일별 평균 따릉이 대여건수(막대그래프)
-월별 평균 따릉이 대여건수(막대그래프)

-장르별 평균 평점(막대그래프)
-영화 전체 평점 분포(히스토그램)
-주요 장르별 평점 분포(바이올린 플롯)


## 사용도구
- Python
- pandas
- numpy
- matplotlib
- seaborn 

## 결과요약
실습을 통해 직접 코드를 적어보고 선,막대,히스토그램, 바이올린 플롯 그래프를 통해
변수들 간의 관계를 보고 해석해봄

## 느낀점
목적에 맞게 다양한 도구를 사용하여 직접 코드를 사용해 보고 
그래프를 만들어 해석해 보는데 의의가 있었다.
